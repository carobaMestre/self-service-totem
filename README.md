## ABOUT

We're introducing our Solution to help a fast food restaurant to provide a quickly self-service to their customers.

## HOW TO SETUP:

In the terminal, hit:

```bash
docker compose up
```

The apps runs into port 3000, it's possible to change the value port or other environments inside a .env file

To access the docs, access:
`http://localhost:3000/docs`
